Benefits of PortaCSS

  • You do not need style element, attribute, or external file.
  • Shorter JavaScript
  • Easy syntax

    • Syntax

    1. Put this code inside of your head:

      			
<link href="https://portacss.eitanalperstein.repl.co/css-files/gens/portacss.css" rel="stylesheet" type="text/css">
    2. To implement, use this syntax:

      		
<element selector="value">
      For example, to change the color of a <p> to red, you would use this code: 
      	
<p color="red">Hello PortaCSS</p>
      Which would output:

      Hello PortaCSS
      NOTE: You can use multiple attributes per element.
      You can also use a different syntax:
